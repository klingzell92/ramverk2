Ramverk 2
====================
Installera
--------------
Börja med att clona repot härifrån Github. Och kör efter det:

```shell
npm install
```
För att köra alla tester så kör man:

```shell
npm test
```

För att starta alla tjänster i docker så kör man:

```shell
 docker-compose up
```
